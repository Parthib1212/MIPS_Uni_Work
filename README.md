# MIPS_Uni_Work

This repository contains MIPS Assembly projects completed for the Computer Architecture Practical course (Summer Semester 2026). The focus of this coursework is on low-level processor logic, stack memory management, the Floating Point Unit (FPU), and pipeline mechanics.

## Repository Structure

Each project is contained within its own directory and includes a dedicated `README.md` explaining its specific logic and technical standards.

* **`Factorial`**: Iterative factorial calculation exploiting delayed branching.
* **`Hofstadter_Sequences`**: Implementation of mutual recursion.
* **`McCarthy-91`**: Deep nested recursive function execution.
* **`Sieve_of_Eratosthenes`**: Prime number identification up to 1000.
* **`Square_Root`**: Floating-point operations using Heron's iteration and bit-level manipulation.
* **`Stack-Memory-String`**: Low-level memory alignment and stack string manipulation.

## Technical Requirements

All programs in this repository are built and tested exclusively for the **MARS 4.5** (MIPS Assembler and Runtime Simulator) environment. 

Please refer to the individual project folders for specific execution instructions, as certain programs require modified simulator settings (e.g., increasing step limits or enabling delayed branching) to run correctly.
